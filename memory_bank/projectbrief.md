# Project Brief: AI-Powered YouTube Automation Tool

## Overview

An automated system that analyzes video content using Gemini AI, generates optimized titles, descriptions, and tags, and uploads the video to YouTube with a custom thumbnail.

## Core Goals

1. **AI Analysis:** Leverage Gemini's vision capabilities for content metadata generation.
2. **Seamless Upload:** Automate the YouTube Data API v3 upload process.
3. **Hybrid UX:** Support both CLI parameters and interactive file selection.
4. **Robustness:** Implement auto-dependency installation, detailed logging, and error handling with retries.
5. **Quality:** Strictly Turkish commit messages and comprehensive logging.
