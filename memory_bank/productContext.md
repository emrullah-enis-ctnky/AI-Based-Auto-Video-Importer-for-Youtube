# Product Context

## Why this project exists?

Content creators often spend significant time generating SEO-friendly metadata and managing the upload process. This tool automates the tedious parts of YouTube management.

## User Experience Goals

- **Zero Setup:** Automatic dependency installation ensures the tool "just works" on any system.
- **Flexible Input:** Power users can use CLI flags, while casual users can rely on the interactive file dialogs.
- **Transparency:** Real-time terminal progress (tqdm) and detailed log files keep the user informed.
- **AI Accuracy:** Using Gemini to ensure titles and descriptions are engaging and relevant.
